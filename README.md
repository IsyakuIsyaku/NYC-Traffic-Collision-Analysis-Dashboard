## PROJECT OVERVIEW

The NYC Traffic Collision Analysis Dashboard is an interactive Power BI project designed to analyze traffic accidents across New York City. The dashboard provides insights into collision patterns, injuries, fatalities, contributing factors, and high-risk locations to support data-driven road safety decisions.

This project transforms raw collision data into actionable insights through interactive visualizations and key performance indicators (KPIs).

Analyze traffic collision trends across NYC.
Identify the most dangerous roads and locations.

Investigate major contributing factors to collisions.
Explore injury and fatality statistics.
Provide insights for improving road safety.

## Key Insights
A total of 238K traffic collisions were analyzed.
January recorded the highest number of accidents.
Friday experienced the highest collision frequency.
Belt Parkway was identified as the most dangerous roadway.
Peak accident activity occurred during busy daytime travel periods.
Driver-related factors were among the leading causes of collisions.
Targeted road safety measures could significantly reduce accidents.
